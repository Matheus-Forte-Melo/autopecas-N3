# Sistema de Gerenciamento de Peças Automotivas

Aplicação Spring Boot desenvolvida para o gerenciamento de catálogo de peças, veículos e suas compatibilidades. O projeto utiliza arquitetura híbrida, fornecendo uma interface Web (Server-Side Rendering) e uma API REST.

Faça os passos comuns para iniciar uma aplicação SpringBoot. Ao iniciar, você terá acesso a documentação via a U.I ou por  [aqui](http://localhost:8080/swagger-ui/index.html) (garanta que você está com o projeto rodando na sua máquina).
**IMPORTANTE: Libere a porta 8080 antes de rodar.**
